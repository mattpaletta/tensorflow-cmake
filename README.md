# tensorflow-cmake

This is a cmake project you can embed in your own projects to use Tensorflow C++, building with Cmake.  (Requires Bazelisk, Python, Numpy, and Bazelisk symlinked with Bazel).

You can see an example project here:
[tensorflow-cmake-example](https://github.com/mattpaletta/tensorflow-cmake-example)

I am not affiliated with Google or Tensorflow in any way.
